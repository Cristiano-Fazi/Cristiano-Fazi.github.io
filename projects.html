Messaging Application


This web application allows users to create accounts and send messages to fellow users. The site uses cookies, both front and back end, to maintain user sessions and log them out after a considerable period of inactivity. The front-end of the website is made entirely in react, while the back-end uses javascript. The database used to store the messages and users was mongoDb. Creating this website greatly increased my understanding of user to user communication as well as allowing to familiarize myself with the highly popular react framework.

FitFolio


Fitfolio is an android application created using kotlin, Jetpack compose and firebase. The purpose of the app is to allow users to plan workouts, keep track of their workouts and generally make life easier for people going to the gym. Users are required to log in in order to access the applications services. Once they've logged in, users can create, delete and update their workout routines as well as follow a routine while exercising. Creating This app allowed me to learn a lot about android and mobile development in general, an essential skill for an software developer.
